# Hazelcast Rust Client

A Rust client library for Hazelcast.

## Project Structure

| Crate | Description |
|-------|-------------|
| `hazelcast-core` | Core types, protocols, and serialization |
| `hazelcast-client` | Client implementation and connection management |

## Building

```sh
cargo build
```

## Testing

```sh
cargo test
```
